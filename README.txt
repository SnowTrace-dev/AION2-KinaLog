AION2 주간 체크리스트 수정본

실행 방법
1. 압축 해제
2. index.html 더블클릭

이번 반영 내용
- 숙제 추가 시 추가한 순서가 아니라 숙제 추가 탭의 고정 순서대로 자동 정렬
- 주간 수익 탭에서 원정 / 초월 / 성역 분리
- 오드 에너지 자동 회복 구현: 3시간마다 15 회복, 최대 840까지 적용
- 주차별 localStorage 저장 구현
- 역대 수익 영역 추가: 몇년 몇월 몇일 ~ 몇월 몇일 주차 / 키나 표시

주의
- 저장 데이터는 브라우저 localStorage에 저장됩니다.
- 다른 브라우저나 다른 PC에서 열면 저장 데이터가 공유되지 않습니다.
