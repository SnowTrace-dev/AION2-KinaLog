AION2 주간 체크리스트 v20

실행 방법
1. 압축 해제
2. index.html 더블클릭

이번 반영 내용
- [성역] 칼드릭스 보상: 오드 에너지 80 소모, 6,000,000 키나 수익으로 변경
- 추가 수익 / 지출 항목 프리셋 추가
  - 항목명을 직접 입력해서 추가하면 다음부터 버튼으로 선택 가능
  - 같은 항목명은 기존 금액에 자동 합산
- 데이터 백업 / 복원 버튼 추가
  - 현재 캐릭터, 숙제, 수익, 지출, 역대 수익, 프리셋 데이터를 JSON으로 저장/복원
- 캐릭터 순서 변경 버튼 추가
  - 캐릭터 카드의 ← / → 버튼으로 순서 변경

주의
- 저장 데이터는 브라우저 localStorage에 저장됩니다.
- 다른 브라우저나 다른 PC에서 열면 저장 데이터가 공유되지 않습니다.
- 복원은 현재 저장 데이터를 백업 파일 내용으로 덮어씁니다.
