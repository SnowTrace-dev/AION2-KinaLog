# 키나로그 V1.1.0

## 실행 주소

https://snowtrace-dev.github.io/AION2-KinaLog/

## 개요

키나로그는 AION2 주간 플레이를 관리하기 위한 개인용 체크리스트 겸 수익 기록 도구입니다.

캐릭터별 오드 에너지, 숙제, 원정, 초월, 성역 보상, 주간 수익, 추가 수익, 지출, 역대 수익을 한 화면에서 관리할 수 있습니다.

## 포함 파일

- `index.html`
- `assets/Logo.png`
- `assets/favicon.ico`
- `assets/favicon.png`
- `assets/Small.png`
- `assets/Large.png`

## 별도 추가 파일

아래 폰트 파일은 직접 `assets` 폴더에 추가하면 적용됩니다.

- `assets/NEXONLv1GothicRegular.ttf`

폰트 파일이 없어도 브라우저 기본 한글 폰트로 정상 실행됩니다.

## 주요 기능

### 1. 캐릭터 관리

- 최대 6개 캐릭터 등록
- 캐릭터 삭제
- 캐릭터 순서 변경
- 전투력 입력 및 수정
- 오드 에너지 / 충전 에너지 직접 수정

### 2. 오드 에너지 관리

- 기본 오드 에너지와 충전 에너지를 함께 표시
- `오드 에너지 000 / 840 (+000)` 형태로 표시
- +10 / +40 충전 에너지 버튼 제공
- 원정 / 초월 / 성역 보상 수령 시 오드 에너지 80 자동 차감
- 오드가 부족하면 보상 수령 방지

### 3. 숙제 체크리스트

- 캐릭터별 숙제 등록
- 일반 숙제는 박스 전체 클릭으로 체크/해제 가능
- 원정 / 초월은 체크 박스와 보상 버튼 중심으로 관리
- 보상 기록이 있는 숙제는 먼저 보상 취소 후 삭제 가능

### 4. 원정 / 초월 관리

- 캐릭터별 원정 / 초월 항목 등록
- 캐릭터별 원정 35회 / 초월 28회 제한 제거
- 원정 / 초월 난이도 선택
- 보상 버튼으로 수익 자동 계산
- 취소 버튼으로 마지막 보상 기록 롤백
- 주간 누적 플레이 보상 횟수 자동 집계

### 5. 성역 관리

- [성역] 루드라
- [성역] 바고트
- [성역] 칼드릭스
- 성역 보상 수령 시 수익 자동 계산
- 칼드릭스는 오드 에너지 80 소모, 6,000,000 키나 수익 기준

### 6. 주간 누적 플레이 보상 조정

- 원정 누적 횟수 표시
- 초월 누적 횟수 표시
- 한 줄 우측 정렬 구조

### 7. 주간 수익

- 원정 수익 집계
- 초월 수익 집계
- 성역 수익 집계
- 각 항목별 판수 표시
- 총 수익 표시

### 8. 추가 수익 / 지출

- 추가 수익 직접 입력
- 지출 직접 입력
- 같은 항목명은 자동 합산
- 프리셋 저장
- 프리셋 삭제

### 9. 역대 수익

- 주차별 수익 자동 저장
- 시즌3 / 챕터1 주차명 표시
- 시즌3 기준 시작일: 2026년 4월 8일
- 챕터1 기준 시작일: 2026년 7월 1일
- 역대 수익 상세 내역 확인 가능

### 10. 데이터 백업 / 복원

- JSON 백업
- JSON 복원
- 백업 파일명: `kinalog_backup_YYYY-MM-DD.json`

## 저장 방식

키나로그는 서버를 사용하지 않습니다.

모든 데이터는 브라우저의 LocalStorage에 저장됩니다.

따라서 다른 PC, 다른 브라우저, 다른 기기에서는 데이터가 자동 공유되지 않습니다.

중요한 데이터는 주기적으로 백업하는 것을 권장합니다.

## 배포 방법

1. 저장소에 파일 업로드
2. `index.html`을 루트에 배치
3. `assets` 폴더를 함께 업로드
4. 필요한 경우 `assets/NEXONLv1GothicRegular.ttf`를 직접 추가
5. GitHub Pages 주소 접속

## V1.1.0 변경 사항

### 버그 수정
- 일반 숙제 항목에서 체크박스뿐 아니라 텍스트 영역 및 박스 영역 클릭으로 체크/해제되도록 수정
- 카드 가로 드래그 기능이 숙제 클릭 이벤트를 방해하지 않도록 수정
- 원정 / 초월 항목은 기존처럼 체크박스와 보상 버튼 중심으로 동작

### 신규 기능
- 상단 날짜 표시를 `시즌/챕터 · N주차 : 날짜 범위` 형식으로 변경
- 상단에 `주간 초기화까지 : 0일 00시간 00분` 카운트다운 추가
- 캐릭터 카드 하단에 캐릭터별 주간 수익 표시 추가

### 캐릭터별 주간 수익 기준
- 각 캐릭터가 이번 주 보상 버튼으로 획득한 원정 / 초월 / 성역 수익을 합산
- 캐릭터 카드 맨 아래 구분선 아래에 표시
